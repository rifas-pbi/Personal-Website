Online Car Services

  • Designed by:Tridz Technologies
  • Developed By:Rifas Ali PBI

  • Handled this entire project alone. Front end is done using basic Html, Css, Bootstrap, Js and Fetch url. Backend is done using Drupal 8.
  • JSON api is used to interact with Drupal to manage GET , POST , UPDATE requests. Created custom apis too by creating custom modules in the backend
  • Used Drupal Commerce to create products , orders , payments etc
  • Integrated Razorpay subscriptions and handled webhooks.
  • Along with various other custom modules subscribed to drupal event subscribers to do certain actions like mail , payment updates etc after order completion.
